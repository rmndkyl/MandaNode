# Requirements 
- CPU: 4 cores
- Memory: 8 GB RAM
- Disk: 300 GB NVME SSD

# One-click Installation
```shell
wget https://raw.githubusercontent.com/rmndkyl/MandaNode/main/AirChain-Nodes/airchains.sh && chmod +x airchains.sh && sed -i 's/\r$//' airchains.sh && ./airchains.sh
```
