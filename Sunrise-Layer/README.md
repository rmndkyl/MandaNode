# One-click Installation
```shell
wget -O run.sh https://raw.githubusercontent.com/rmndkyl/MandaNode/main/Sunrise-Layer/run.sh && chmod +x run.sh && sed -i 's/\r$//' run.sh && ./run.sh
```
